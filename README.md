## Use Docker

Run api service and client app with Docker Compose

```sh
docker-compose up
```

### API Service / Client App

- ./packages/server-api
- ./packages/client

Install dependencies

```sh
npm install
```

Test

```sh
npm test
```

Run dev

```sh
npm start
```

- http://localhost:3001/ - api service
- http://localhost:3000/ - client app
